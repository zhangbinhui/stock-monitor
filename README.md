# 📊 高管增持三重过滤监控系统

每日自动筛选A股高管集中增持的公司，通过**陈老师三重过滤体系**评估，发送邮件报告。

## 三重过滤体系

| 过滤层 | 内容 | 说明 |
|--------|------|------|
| 第一重 | **高管增持** | ≥5位高管通过竞价交易买入（排除大股东增持） |
| 第二重 | **基本面估值** | 按股票类型评估：价值股看扣非PE-TTM，成长股看PEG，周期股看PB+利润趋势 |
| 第三重 | **均线择时** | MA10/20/30/60多头排列=买入，空头排列=回避 |

## 报告内容

- 📈 **宽基ETF量价监控** — 沪深300/中证500/创业板/科创50/恒生科技
- 🏢 **高管增持明细** — 增持人数、金额、市值、年薪比
- 📊 **基本面分析** — 扣非PE-TTM、PB、ROE、利润趋势、股票分类
- 📉 **技术面+投资建议** — 均线状态、操作信号、三重过滤结果
- 📢 **最新公告动态** — 业绩预告、回购、增减持、诉讼等
- ⚠️ **卖出信号监控** — 均线破位、基本面恶化、止损提醒

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置邮箱
cp .env.example .env
# 编辑 .env 填写SMTP信息

# 配置订阅者
cp subscribers.example.json subscribers.json
# 编辑 subscribers.json 添加收件人

# 手动运行
python main.py

# 定时任务（每个交易日早8点）
crontab -e
# 添加: 0 8 * * * /path/to/venv/bin/python3 /path/to/main.py >> /path/to/logs/cron.log 2>&1
```
